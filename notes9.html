  • Lecture 208/9: selecting with jQuery
    1. jquerydemo.html
    • note: to access same output as colt, use $('h1')[0]; - otherwise you get the jQuery object, not the HTML element (what you want)
    • use $ - which functions like querySelectorAll
    • to change the property of the thing you selected, use the method .css(property, value):
    • ex: $("#adorable").css("border","2px solid red")
    • without jQuery: document.querySelector("#adorable").style.border = "2px solid red"
    • jquery css is more powerful: you can make an object (ex: styles) and define multiple styles to pass into .css(styles)
      • var styles = { color: "red", background: "pink", border: "green 2px dashed" }
      • $("h1").css(styles)
    • you can also select and apply styles to multiple elements - without needing forloops
      • $("li").css("color", "blue") would make all li's blue
    • note that css uses font-size, but js uses fontSize
    • $ returns everything in a list, even if there's only one thing selected

Advanced jQuery

  • Lecture 214: click events
    • the 3 most important ones: click(), keypress(), on()
    • ex: $('button').click(function(){...})
    • ex: $('h1').click(function(){ alert("h1 clicked"); })
    • refering to the element that was clicked: jQuery version of 'this': $(this)
  • Lecture 216: keypress
    • fires in between key was pressed up and down
    • there's also keyup and keydown, but keypress is the most common
    • you can listen for a keypress but only do something when the use presses enter (for example)
    • "which" is the keycode for a keypress - for enter, it's 13
    • $('input').keypress(function(event){ if(event.which === 13) { alert('you hit enter'); } })
  • Lecture 217: on
    • can take the place of all the methods above
    • need to select something, then call on. need to specify what kind of event we're listening for
    • Ex to change the clicked h1 to purple: $('h1').on('click', function(){ $(this).css("color", "purple"); })
    • hover: "mouseenter" instead of click
    • but css has pseudoselector for hover, so typically you wouldn't do this with js
  • Lecture 218: jQuery effects
    1. jqueryeffects.html
    2. effects.js
    • fading three divs in and out
    • fade: fadeIn, Out, Toggle
    • slide: slideUp, Down, Toggle
    • all set display: none at the end
    • remember that you have to put things in the callback function otherwise they won't wait for the fade to finish
