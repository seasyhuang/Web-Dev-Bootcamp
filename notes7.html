  • Lecture 125: boolean logic
    • true, false, and beyond
    • boolean logic - writing statements that evaluate to true or false
    • comparison operators: > >= < <= == != === !==
    • === does not perform type coercion (takes both vars and tries to turn them into same type), while == does
      • 5 == 'x' is true
    • interesting cases of ==
      • true == "1" is true
      • false == 0 is true
      • NaN == NaN is false
