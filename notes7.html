  • Lecture 143-149: introduction to functions

    2. function.js
    • two steps: declare then call - a variable that stores lines of code
    • declaring the function: function fName(){ do something }

    • functions with arguments: function fName(argument1, argument2){ do something with argument2 }

    • the return keyword: output of the function
    • in the chrome developer console, there will be a left arrow < on the side of the returned value
    • useful functions from example:
      • str.charAt(0) returns first character
      • str.slice(1) returns everything including and after str at index ("hello".slice(1) --> ello)

JS Arrays

  • Lecture 154: introduction
    • data scructure
    • define: var friends = ["fds", "rfdjkwl", "dfjkla"];
    • adding new info to array: can just say friends[3] = "new friend", code won't break if you go larger than array size
    • if you add something to an array (size 3) at index 10, everything between would be undefined
    • not type specific
    • to get length: array.length
  • Lecture 155/6: array methods
    • methods: push, pop, shift, unshift
    • adding to end of array: array.push(newThing), returns the length of array
    • removing from end of array: array.pop(), returns the removed item
    • adding to start of array: array.unshift(newThing), returns the length of array
    • removing from start of array: array.shift(), returns the removed item
    • finding something in your array: array.indexOf(what you're looking for), returns index of first instance or -1 if not there
    • copy different portions of array: array.slice(1,3) - includes 1, not 3
      • nums = [1, 2, 3, 4]
      • nums.slice(1,3) --> [2, 3]
      • nums.slice(1,2) --> [2]
      • nums.slice(1) --> [2, 3, 4]
