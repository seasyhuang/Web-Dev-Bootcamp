  • Lecture 90: changes from bootstrap 3.x to 4.x
    • we usually have the documentation open anyway while we're coding, so this is not a line by line comparison
    • changes are listed here
    • flexbox is enabled by default
      • bootstrap is now built around flexbox
      • makes things cleaner
    • rems are now the default (instead of px)
    • global font size is 16px instead of 14px
    • there are now 5 tiers in grid system
    • new utility classes (didn't exist in bs 3)
    • new component called cards
    • no more glyphicons - if you want to use icons, use external (like fontawesome)
    • everything kind of renamed - ex. no more btn-small class, no more btn-default class

  • Lecture 93: colors and backgrounds

  • Lecture 94: typography
    1. typography.html
    2. bootstrap typography
    • no more page header class
    • new display classes - a larger, slightly more opinionated heading style
    • blockquotes are now div classes
    • in the p class: p class=mb-0 is optional, sets the margin bottom to 0
    • we now have a footer class: footer class="blockquote-footer"
      • citation inside the footer class class: cite title="this will be italicized"
    • right align: text-right
    • big change: font units
      • from px to rems - root ems
      • if the root font size changes, everything with rem will become bigger
  • Lecture 95: new features in bootstrap 4
    1. utilities.html
    2. bootstrap utilities
    • borders in spans: span class="border" you can now add and subtract borders
    • to selectively subtract a border: border-top-0
    • no way to easily subtract border thickness, but you can change border radius and color
    • spacing: (not responsive)
      • new syntax: m for margin, p for padding
      • specify by adding either t, b, l, r for top bottom left right
      • x does padding left and right, y does passing top and bottom
      • examples: p-0, pt-5
      • 5 levels of padding, based of rem
      • same applies to margins m
  • Lecture 96: responsive breakpoints
    1. breakpoints.html
    2. bootstrap breakpoints
    3. bootstrap spacing
    • responsiveness
    • new screensize: extra small size for portrait mode phones (small size is landscape mode on phones)
    • to incorporate responsiveness, use this format: {property}{sides}-{breakpoint}-{size}
    • example: button class="btn p-sm-5 p-md-0 p-lg-5 p-xl-0"
    • this p-sm-5 would affect everything small and above
    • note that xs is the default - set it with p-0 for example
    • in the example, the margins get bigger as the page gets bigger
  • Lecture 97: navbars
    1. navbar4.html
    2. bootstrap navbars
    • now you can change at what point the navbar collapses (technically, when it expands)
    • entirely rewritten in flexbox
    • new required class: .navbar-expand{-sm|-md|-lg|-xl}
    • no more navbar-default, now it's dark and light
    • you also have to set a background
    • no more autromatic margin-bottom
    • for dark navbars: nav class="navbar navbar-expand-xl navbar-dark bg-info" the navbar-dark refers to light text, and bg-info specifies the color of the background
  • Lecture 98: new display utility
    1. displayutility.html
    2. bootstrap display utility
    • there used to be hidden and visible classes that changed what was visible on what page size (responsive)
    • those were removed: now there is the d utility (display)
    • example: .d-{value} for xs size
    • hide me on xl: h1 class="d-xl-none"
    • always hidden except large: h1 class="d-none d-lg-block d-xl-none"
      • from xs (default) up, nothing. when we get to large, show. when xl, hide it again
