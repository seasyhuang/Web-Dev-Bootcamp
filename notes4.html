  • Lecture 77: navbar
    1. navbar.html
    2. bootstrap.css (downloaded)
    • navbars
    • simplest way of making a navbar: start with the nav element, and two classes: navbar navbar-default
      • need a div inside the nav with class="navbar-header"
        • inside that, need anchor tag with brand (clicking it would take you to homepage) < a class="#" class="navbar-brand">< /a>
    • next, need a second div/ul element with class="nav navbar-nav"
      • for every item in this div/ul --> new li with a tag inside to link to new page
      • it's more conventional to use an ul - works fine with div though
    • adding something to the right side of the navbar:
      • need an element (div or ul) with class="nav navbar-nav navbar-right"
      • if there is no container around the overall navbar, the elements on the right will be right up against the page edge
      • container should be inside the navbar --> content becomes restricted
      • if the container is outside the navbar, whole navbar is restricted
    • looking at the default navbar
      • if you don't add the bootstrap js file/cdn to your html, the dropdown buttons won't do anything
      • need to include jquery cdn as well
      • included at the bottom of the body
      • jquery cdn must come before js because the javascript relies on jquery
      • dropdowns should work now
      • look at div class="collapse navbar-collapse" - content will collapse when the browser hits mobile size
      • to get the hamburger: see the section starting with button type="button" class="navbar-toggle collapsed" ...
        • this needs to be placed in the navbar header before the Koffee a tag
        • button, inside is 3 spans: each span is one of the narrow bars
        • showing and hiding the content depends on the data-target attribute in the button and the id in the collapse section
        • these have to match (#data-target vs id (no #))
        • in example, i called it datatargetandid
  • Lecture 78/79/80: bootstrap grid system
    1. grid.html
    2. grid2.html
    • the main reason people use bootstrap
    • grid system
    • any time you use the grid system, needs to be in a container class
    • inside the container, you need a row
    • grid is made of 12 units
    • 2x col-lg-6 = 12
    • sizes:
      • there are four sizes: xs, sm, md, lg
        • xs - phone
        • sm - tablet
        • md - desktop
        • lg - bigger desktops (>1200px)
    • example: class="col-lg-1 col-md-2" when the grid is large size, take up 1 unit; medium size, 2 units
    • we may want to do this: col-lg-3 col-md-3; however, you don't need to put the lg if it's the same as md col-md-3
    • adding another row inside a row splits it into another 12 units
