  • Lecture 177/8: important selector methods
    • select IDs: getElementById() - returns an object (looks like html in the console)
    • select class: getElementsByClassName() - note that it's elements, returns a list of elements (not array)
    • select by tag name: getElementsByTagName() - selects li, ul, div, head, body, etc. and returns as a list (even if there is only one element in the list)
    • note that querySelectorAll gives all matches of the inputted css selector, but querySelector gives the first instance

  • Lecture 180/1: manipulating style
    • changing element style:
      • huge object
      • example (after selecting, set var to h1): h1.style.border = "pink"
      • note that right side has to be a string
    • useful way of applying a lot of changes at once:
      • first define a class in your css
      • then, you can apply to a variable (ex. thing) you've selected by using:
        • to add: thing.classList.add("new-css-class")
        • to remove: thing.classList.remove("new-css-class")
        • to switch on or off: thing.classList.toggle("new-css-class")
      • try it out on this line var tbd = document.getElementById("tbd"); tbd.classList.toggle("big");
      • would be useful for todo list, for example
    • Lecture 182: manipulating text and content
      • text content - get all the text in an element of a variable using myVar.textContent
      • set it: myVar.textContent = "blah";
      • try it out on this line var t = document.getElementById("textex"); t.textContent = "blah"; - this completely overwrites it
      • to get the html content plus text: myVar.innerHTML
      • try: var t = document.getElementById("textex").innerHTML; - gets you text and html in the variable t
      • another difference between textContent and innerHTML - if i wanted to add html elements, can't use textContent - it will treat it as normal text, won't render the html
        • try with textContent (need to remove spaces around h1): document.getElementsByClassName("text2")[0].textContent = "< h1>hi< /h1>"
        • try with innerHTML (need to remove spaces around h1): document.getElementsByClassName("text2")[0].innerHTML = "< h1>hi< /h1>"

    • Lecture 184/5: playing with google codealong
      • In the following lecture, Playing With Google Code Along, you'll be changing the logo on the google website using the src attribute. However, this method no longer works. Instead, use the srcset attribute to change the logo image.
      • have to use for loops, can't use for each
      • you can use a loop to change multiple things:
        • get all the links on page: links = document.querySelectorAll("a")
        • print all textContent of the links: for (var i = 0; i < links.length; i++) { console.log(links[i].textContent); }
        • get all link destinations of links: for (var i = 0; i < links.length; i++) { links[i].getAttribute("href"); }
