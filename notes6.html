  • Lecture 99: Flexbox and layout
    1. flexbox.html
    • Flexbox
    • exists outside of the BS world: a model for positioning content inside of a box
    • but with BS, you can do the same thing but more easily
    • first, you need to make whatever div container you're working with into a flexbox: div class="d-flex"
    • the start of the flexbox/the main axis is from left to right
    • the cross axis is top - bottom
    • if we want to easily move all the content to the right side of the flexbox, we can use justify-content-end
      • example: div class="border border-dark d-flex justify-content-end"
      • we can also do center justify-content-center
      • to put all the space between the objects: justify-content-between
      • to put all the space around the objects: justify-content-around
    • what if we don't want the items to take all the space? align-items-stretch is the default
      • start, end, center, baseline, or stretch (browser default)
    • all of this is responsive
    • justify: main axis
    • align: cross axis
  • Lecture 100: Flexbox 2
    1. flexbox.html
    • to easilly flip the direction of items: add the class flex-row-reverse
    • you can even transpose (rows to columns) by changing flex-row to flex-column
      • align items now refers to left to right
    • flip it all with flex-row-reverse
    • example: div class="d-flex flex-column flex-md-row justify-content-between
      • setup flexbox: d-flex
      • default (from smallest size) is a column: flex-column
      • at medium size and above: turn into a row: flex-md-row
      • put space in between elements: flex-md-row justify-content-between

  • Lecture 105: the grid and flexbox
    1. gridflexbox.html
    • easy way to add spacing between elements using both the grid (col-) and flexbox (justify, align, etc)
    • you can have 3 divs with col-3 each, then play around with justify- or align- to change the leftover 3 units of spacing
